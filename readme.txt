== Description ==

**BestPrice.gr XML Feed for WooCommerce** is WordPress plugin that generates XML feed according to bestprice.gr specs

**If you like this plugin please support the developers and/or rate it at WordPress.org**

#### Features

* Supports fashion products (shoes, clothes etc)
* Supports bookstores (ISBN field)
* Fully customizable

#### Requirements:

* WordPress version: 3.5.1+
* Apache version: 2.1+
* PHP version: 5.4+

== Installation ==
